Using inverse format to find the solution
### Step 4: 
Print result.

## Program:
```python
#Program to find the inverse of a matrix.
#Developed by:yuvabharathib
#RegisterNumber:22002787
import numpy as np
a=np.array([[1,0,3],[-1,2,-2],[2,3,-1]])
inverse=np.linalg.inv(a)
print(inverse)
```
